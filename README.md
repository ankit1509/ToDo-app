## Running the project
***

1. Clone the GitHub repository.

    ```bash
    git clone https://github.com/ankit1509/ToDo-app.git
    ```

2. Install the project.

    ```bash
    cd ToDo-app
    npm install
    ```

3. Start the project. 

    ```bash
    npm start
    ```

A web browser should automatically open [http://localhost:3000](http://localhost:3000).
